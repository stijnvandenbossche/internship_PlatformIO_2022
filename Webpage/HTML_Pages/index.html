PlatformIO

PlatformIO is a tool designed for systems engineers and software developers, to write applications for embedded products. It works across different operating systems, and for different platforms using a variety of architectures.

To develop this project, it was used as a plug-in for Visual Studio Code.

PlatformIO automatically downloads and configures necessary tools to build and upload projects.

Website made by Stijn Van den bossche during internship at Verhaert for a bachelor's thesis at Thomas More.
